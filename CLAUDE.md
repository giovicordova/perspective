# Perspective

Claude Code plugin distributed as a standalone repo.

## Structure
- `skills/perspective/SKILL.md` — the skill (this is the product)
- `.claude-plugin/plugin.json` — plugin manifest
- `.mcp.json` — bundles Context7 MCP dependency
- `README.md` — install instructions and usage
- `VISION.md` — design philosophy

## Working on this project
- Changes to SKILL.md are the primary deliverable
- Keep SKILL.md under 500 lines
- Test by installing the plugin locally and invoking /perspective
